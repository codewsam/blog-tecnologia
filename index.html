Titulo da postagem 1

postado em 20 de março

Um computador é um equipamento que processa dados eletronicamente. Esses dados são informações que podem ser programadas para diversas finalidades. Então qualquer maquina que cumpra esses requisitos são computadores. Os computadores usados nas casas e escritórios, são os desktops ou computador de mesa. Já para necessidades de transportar o computador, existe o notebook. Todos esses tipos possuem componentes indispensáveis como processador, memória RAM, placa mãe entre outros.
